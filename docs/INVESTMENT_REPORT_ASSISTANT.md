## 一、项目背景

### 1.1 业务场景

投资公司员工负责投前尽调和投后管理，需要定期汇报项目真实状况和风险点，供经理/老板做投资决策。

**核心痛点：**
- 员工汇报可能有意无意忽略关键信息
- 风险点描述模糊，缺乏量化
- 问题只摆在那里，没有解决方案
- 决策者难以快速获取关键信息

### 1.2 目标

通过 AI 多轮追问，挖掘项目真实状况和风险点，生成结构化报告，支持投资决策。

**核心价值：**
- 补齐员工忽略的关键信息
- 深挖风险根因和影响
- 推动形成解决方案
- 输出决策所需的关键信息

## 三、汇报类型

| 类型 | 触发方式 | 适用岗位 | 目的 |
|------|----------|----------|------|
| 出差计划 | 出差审批通过后提醒 | 所有岗位 | 明确出差目的、目标、方案 |
| 出差总结 | 出差结束后提醒 | 所有岗位 | 记录出差成果，识别问题 |
| 投前尽调 | 出差总结后触发 | 投资经理、法务、财务 | 评估投资价值，识别风险 |
| 投后管理 | 每周五17:00提醒 | 投资经理、项目经理 | 跟踪进展，识别风险 |

**出差闭环流程：**

```
出差审批通过 → 出差计划（目的/目标/方案/协助）
      ↓
   出差执行
      ↓
出差结束 → 出差总结（成果/问题/风险/后续）
      ↓
投前尽调/投后管理（深入分析）
```

## 四、追问维度设计

### 4.1 投前尽调 - 投资经理

| 轮次 | 追问维度 | 具体问题 |
|------|----------|----------|
| 第1轮 | 项目基本面 | 项目/公司做什么？市场规模多大？团队背景如何？核心人员履历？ |
| 第2轮 | 财务与业务 | 财务数据如何？营收/利润/现金流？业务数据可验证吗？关键指标？ |
| 第3轮 | 风险识别 | 最大的风险是什么？法律合规？政策风险？竞争风险？ |
| 第4轮 | 风险量化 | 风险发生的概率？影响程度？有什么缓解措施？最坏情况？ |
| 第5轮 | 决策建议 | 你的判断：投/不投/观望？依据？关键决策点？需要什么支持确认？ |

### 4.2 投后管理 - 投资经理

| 轮次 | 追问维度 | 具体问题 |
|------|----------|----------|
| 第1轮 | 项目进展 | 本周期里程碑完成情况？与预期对比？超前/滞后多少？ |
| 第2轮 | 经营状况 | 关键经营指标变化？财务状况？现金流？团队稳定性？ |
| 第3轮 | 问题识别 | 遇到什么问题/困难？新出现的风险？与预期不符的地方？ |
| 第4轮 | 原因分析 | 问题根本原因？执行问题还是外部变化？团队应对能力？ |
| 第5轮 | 投资建议 | 投资建议：追加/持有/退出？需要什么支持？谁协助？ |

### 4.3 出差计划 - 所有岗位

| 轮次 | 追问维度 | 具体问题 |
|------|----------|----------|
| 第1轮 | 出差目的 | 这次出差主要目的是什么？要去见谁？考察什么？ |
| 第2轮 | 目标设定 | 想达成什么具体目标？怎么判断出差成功？ |
| 第3轮 | 行动方案 | 准备怎么达成目标？有什么具体计划？ |
| 第4轮 | 资源需求 | 需要什么支持/协助？需要谁配合？ |
| 第5轮 | 风险预判 | 可能遇到什么困难？有什么备选方案？ |

### 4.4 出差总结 - 所有岗位

| 轮次 | 追问维度 | 具体问题 |
|------|----------|----------|
| 第1轮 | 出差目的 | 这次出差主要目的？见了谁？ |
| 第2轮 | 关键发现 | 最重要的发现是什么？ |
| 第3轮 | 问题风险 | 发现什么问题/风险？ |
| 第4轮 | 后续行动 | 需要跟进什么？谁负责？ |
| 第5轮 | 结论建议 | 你的判断和建议？ |

### 7.1 出差闭环流程

```
┌─────────────────────────────────────────────────────┐
│  员工提交出差审批                                     │
└────────────────────┬────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────┐
│  审批通过 → 创建出差记录                              │
│  发送消息："请制定出差计划"                           │
└────────────────────┬────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────┐
│  员工回复"开始" → 5轮追问 → 生成出差计划              │
│  保存到汇报记录，更新出差记录.计划状态=已完成          │
└────────────────────┬────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────┐
│  出差执行中...                                        │
└────────────────────┬────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────┐
│  出差结束（结束时间+1天）→ 发送消息                   │
│  "请提交出差总结"                                     │
└────────────────────┬────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────┐
│  员工回复"开始" → 5轮追问 → 生成出差总结              │
│  保存到汇报记录，更新出差记录.总结状态=已完成          │
└────────────────────┬────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────┐
│  发送给上级                                           │
└─────────────────────────────────────────────────────┘
```

## 八、
触发机制

### 8.1 出差审批触发

```python
# 出差审批通过后触发
async def on_trip_approved(approval):
    """出差审批通过 → 触发出差计划"""
    user_id = approval.user_id
    project_name = approval.form_data["项目名称"]
    start_date = approval.form_data["开始时间"]
    end_date = approval.form_data["结束时间"]
    
    # 创建出差记录
    trip_id = await create_trip_record({
        "user_id": user_id,
        "project_name": project_name,
        "start_date": start_date,
        "end_date": end_date,
        "approval_id": approval.instance_id,
        "plan_status": "未制定",
        "summary_status": "未提交"
    })
    
    # 发送提醒
    await send_message(
        user_id,
        f"✈️ 出差审批已通过\n\n"
        f"项目：{project_name}\n"
        f"时间：{start_date} ~ {end_date}\n\n"
        f"请先制定出差计划，回复「开始」继续"
    )
    
    # 设置出差结束提醒
    await schedule_reminder(
        trigger_time = end_date + timedelta(days=1),
        task_type = "trip_summary",
        trip_id = trip_id
    )
```

### 8.2 周报提醒

# 每周五 17:00 触发
async def weekly_reminder():
    """周报提醒"""
    employees = get_active_employees()
    
    for emp in employees:
